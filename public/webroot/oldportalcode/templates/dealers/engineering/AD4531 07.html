Engineering Note #4531_07

Equipment - AD4531 Indicator Options

Problem - Lack of fitting details in the Instruction Manual

Detail:

There have been several reports of indicator options not functioning after installation by customers. The instruction manual details the function settings for each option but does not address the setting o on board links and switches. Failure to set these items, to match the option type, will result in the incorrect or non-operation of the option.

The details are shown below:

Option AD4531 OP-04 or OP-05

Referring to the drawing below carry out the following steps:-

  1. Cut JP1 and JP2
  2. Set Switch S1 to OFF
  3. Plug in the option to connector J3
  4. Secure the option using the M3 x 6 screw

Option AD4531 OP-07

Referring to the drawing below carry out the following steps:-

  1. Cut JP1
  2. Set Switch S1 to ON
  3. Plug in the option to connector J3
  4. Secure the option using the M3 x 6 screw
