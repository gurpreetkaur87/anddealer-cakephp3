Marketing Tip

Subject - Year 2000 Compliance

There are 3 distinct types of problem that may be faced in the year 2000:-

1. If the product is keeping track of the year only by use of the lower two digits, then in the year
2000 A.D. a calculation made in connection with the date may be in error.

2. If the product is keeping track of the year only by the two lower digits then the higher digits
are automatically set to `19' and the date cannot change to 2000 A.D.

3. The year 2000 A.D. is a leap year but the product cannot set February 29th.

None of the A&D products that use a clock function carry out calculations based upon the date so
problem 1 is not relevant.

All of the company's products have been examined to establish the type of clock circuit used in t
hose that have a clock function. The use of these in the different types of device is explained.

Type A

There is no year in the clock function and the year is set by the user. Since the leap year is
not considered in this function (there is no year in the calendar function so it cannot set a leap year)
the next day to February 28th is March 1st, but it is possible for the user to set Febraury 29th.
Similarly it is possible to set the year 2000, but the input and the printing are set only by the two
lower digits. The products that have this type of clock function may have one of two IC types.
The UC:D1990C, which does not have a year function or the UC:D4990C, used as a replacement
for the UC:D1990C, which does have the year function but the product does not use it.

Type B

The calendar function in these products uses an IC with a year function, UC:62421A or
UC:62423A, but the year is only tracked by the lower two digits. Thus it automatically
keeps track of the year and the leap year. The year 00 is a leap year. The display and
printing of the year is only two digits thus the day after 31/12/99 is 01/01/00.

Type C

These products use the same type of IC as case Type B above and the IC is programmed
to keep track of changing the year from 1999 A.D. to the year 2000 A.D. There is no problem
with these type of products.
