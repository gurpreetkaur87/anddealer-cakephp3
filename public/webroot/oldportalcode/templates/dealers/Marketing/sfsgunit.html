Marketing Tip

Subject - SF/SG - Unit Price Hold

  Please take note that the SF/SG series do have a Unit Price HOLD facility. It is not obvious that this facility exists unless you fully study the Function settings on page 11 of the Instruction Manual version 1.1.
  Function F2 - Cancellation of unit price is the secret.
Set this to value 0 (Selectable) The unit price can be held or cancelled by pressing the CHANGE key to select the required mode.
Set this to value 1 The unit price is held until either a new price is entered or the power is turned off.
Set this to a value 2 The unit price will always be cancelled
