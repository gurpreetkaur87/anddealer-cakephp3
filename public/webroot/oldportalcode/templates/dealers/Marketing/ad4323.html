Marketing Tip

Subject - AD4323DR - Dual Range Indicator

  The AD4323DR digital indicator was developed from the standard AD4323
model to provide, principally, an indicator suitable for connection to a
weighbridge and to provide a capacity in excess of 60 tonnes but retaining
the 0.02 tonne graduation in the lower range.
  The software changes were minimal to achieve this function and the standard
AD4323 Instruction Manual is used, in conjunction with a modification sheet
applicable to the AD4323DR. This modification sheet causes some heartache
for installers in its interpretation so some explanation is required.

The Changes from the Standard AD4323

Function F1- Decimal point position This function has been removed since the decimal point position is
set automatically by the modified function F2.
Function F2 - Weighing Unit F2=1 The weighing unit is kg and there is no decimal point

F2=2 The weighing unit is tonne and the decimal point is in the second position i.e. x.xx

When F2=1 A dummy zero is placed by the software at the end of the display and this must be taken into account during calibration.

Function F11 - Internal parameter This function must be set to a value 1 to enable the AD4323DR to function correctly.
Function F65 - Low Range Capacity This function determines the point at which the indicator switches from low to high range graduation. The installer has complete control over this value. However if you have selected F2=1 a dummy zero has been inserted by the software and you should enter (Low Range capacity divided by 10) as the capacity.
Function F66 - Internal parameter This function must be set to a value 0 to enable the AD4323DR to function correctly.
Calibration

In the standard calibration routine the Minimum Division that is entered is the Low Range division. If function F2 has been set to value 1 the display will show 1, 2 or 5 as the division but because of the dummy zero these figures will actually represent 10, 20, 50. The minimum division for the High range will be automatically allocated by the software as the next higher increment i.e.

Lo division Hi division
1 2
2 5
5 10
10 20
20 50

In the calibration routine the maximum capacity required to be entered is the High range capacity. Note that if F2=1 the dummy zero will be applicable and you should enter (Hi max divided by 10) as the Hi range capacity.

Example 1: Required 80t max; Low Range to 60t. Minimum Division 0.02t

F2=2; F11=1; F65=6000 (60.00); F66=0. Calibration d=2 (Low range); Maximum = 8000 (80.00).

Example 2: Required 80,000kg max; Low Range 60,000kg; Min. Div. 20kg

F2=1; F11=1; F65=6000 (6,0000/10); F66=0. Calibration d=2 (20/10); Maximum = 8000 (80,000/10).
